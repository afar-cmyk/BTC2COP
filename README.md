# BTC2COP
Currency converter calculator and chat bot for Cisco Webex

## Front-end
Nuxt3 and Tailwind CSS hosted on Netlify\
<br>

## Web scraper
Puppeteer using Serverless Lambda Functions hosted on Vercel\
<br>

## Database
MongoDB stored on MongoAtlas\
<br>
